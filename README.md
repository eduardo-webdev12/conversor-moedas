## 📖 Sobre o Projeto

Este é um **Conversor de Moedas** que desenvolvi para praticar meus conhecimentos em **TypeScript** e **JavaScript**. 

O projeto consome uma API de câmbio em tempo real e oferece uma interface moderna e responsiva para converter valores entre 9 moedas principais.

### 🎯 Por que fiz este projeto?

- Queria criar algo **útil** que pudesse usar no dia a dia
- Precisava praticar **consumo de APIs** com JavaScript
- Queria aprender **TypeScript** na prática
- Precisava de um projeto para meu **portfólio**

---

## ✨ Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| 💰 **9 Moedas** | BRL, USD, EUR, GBP, JPY, CAD, AUD, CHF, CNY |
| 🔄 **Conversão em Tempo Real** | Integração com API de câmbio |
| 📊 **Taxa de Câmbio** | Mostra a taxa utilizada na conversão |
| 📋 **Histórico** | Salva as últimas 10 conversões no localStorage |
| 🔁 **Troca Rápida** | Botão para inverter as moedas |
| 📱 **Responsivo** | Funciona em celular, tablet e desktop |
| 🎨 **Design Moderno** | Interface limpa e intuitiva |

## 🚀 Como Executar o Projeto

### Pré-requisitos
- Node.js instalado
- NPM ou Yarn

### Passos

```bash
# 1. Clone o repositório
git clone https://github.com/eduardo-webdev12/conversor-moedas.git

# 2. Entre na pasta
cd conversor-moedas

# 3. Instale as dependências
npm install

# 4. Compile o TypeScript
npm run build

# 5. Execute o servidor
npm run serve

# 6. Acesse no navegador
http://localhost:3000
